## 🔄 ЦИКЛ РАБОТЫ: 3-месячный блок

```
МЕСЯЦ 1: НАСТРОЙКА
├─ Неделя 1
│  ├─ Взрослый читает методологию
│  ├─ Первое эссе "Зачем мне"
│  ├─ 1–2 пробных эпизода
│  └─ Первое наблюдение состояния
│
├─ Неделя 2–3
│  ├─ Выбираешь 1 различение + 1 контекст
│  ├─ 2–3 эпизода в неделю
│  ├─ Ведёшь еженедельный лог
│  └─ Наблюдаешь состояние ребёнка
│
└─ Неделя 4
   ├─ Первый месячный обзор
   ├─ Масти-мап по 4 различениям
   ├─ Рефлексия взрослого
   └─ Решение: продолжаем? углубляем? пауза?

МЕСЯЦ 2: РАСШИРЕНИЕ
├─ Неделя 5–6
│  ├─ Добавляем 2-й контекст (вариативность)
│  ├─ Или углубляем то же самое
│  ├─ 3–4 эпизода в неделю
│  └─ Смотрим перенос в жизнь
│
├─ Неделя 7–8
│  ├─ Если перенос есть → может добавить 2-е различение
│  ├─ Или расширить контексты
│  ├─ Продолжаем логи
│  └─ Проверяем, не перегруз ли
│
└─ Неделя 9–10 (по сути неделя 1)
   ├─ Масти-мап обновляем
   ├─ Рефлексия месяца 2
   └─ Решение на месяц 3

МЕСЯЦ 3: СТАБИЛИЗАЦИЯ + ПЕРЕНОС
├─ Неделя 11–12
│  ├─ Может быть, 2-е или 3-е различение
│  ├─ 4–5 контекстов (максимум)
│  ├─ Фокус: перенос в естественные ситуации
│  └─ Логи + наблюдение
│
└─ Неделя 13–14 (конец квартала)
   ├─ Финальная Mastery Map
   ├─ Итоговое эссе взрослого
   ├─ Что прижилось? Что нет?
   ├─ Где видели спонтанный перенос?
   └─ Решение: продолжаем блок 2? пауза? интеграция в жизнь?
```

## ✅ ПРОЦЕССНЫЕ ГЕЙТЫ (контрольные точки)

| Гейтoб | Когда | Что проверить | Что дальше? |
|--------|------|---|---|
| **День 1** | Сразу | Согласованность взрослых, есть ли место для логов | Вперёд / Пауза если рассинхрон |
| **Конец недели 1** | После 1–2 эпизодов | Получилось ли? Ребёнок был в ресурсе? | Вперёд / Упростить формат |
| **Конец недели 2** | После регулярности | Ребёнок готов? Видни ли первые признаки? | Вперёд / Менять контекст или взрослого |
| **Неделя 4** | Конец месяца 1 | Mastery Map по 4 различениям; есть ли хоть минимум? | Углубить / Пауза если 0 прогресса |
| **День 30** | Конец месяца | Рефлексия взрослого; видит ли изменения? | Вперёд на месяц 2 / Переоценить подход |
| **День 60** | Конец месяца 2 | Перенос начался? Спонтанное применение? | Вперёд / Концентрировать на качестве, не количестве |
| **День 90** | Конец квартала | Итоговые метрики; 70%+ спонтанного переноса? | Повторить блок с новым фокусом / Интегрировать в жизнь |

## 📊 КОМПОНЕНТЫ И ИХ ВЗАИМОДЕЙСТВИЕ

### Компонент 1: РАЗЛИЧЕНИЕ (из 01-contract)
**Функция:** Точное описание что это такое  
**Входит в:** Методологию, логи, карты  
**Выход:** Ребёнок применяет в жизни

### Компонент 2: РОЛЬ (из 02-entities)
**Функция:** Кто что делает  
**Входит в:** Явный контракт между взрослыми  
**Выход:** Согласованная работа

### Компонент 3: МЕТОД (из 03-methods)
**Функция:** Пошаговый протокол эпизода  
**Входит в:** Каждый эпизод с ребёнком  
**Выход:** Регулярные, качественные эпизоды

### Компонент 4: ЛОГ (из 04-products)
**Функция:** Фиксация наблюдений  
**Входит в:** Каждый эпизод (3 строки) + месячный обзор  
**Выход:** Данные для корректировки

### Компонент 5: КАРТА (Mastery Map)
**Функция:** Видеть прогресс по 4 различениям  
**Входит в:** Ежемесячный обзор  
**Выход:** Решение, что делать дальше

### Компонент 6: ЗАЩИТА (из 05-failure)
**Функция:** Вектор от типовых ошибок  
**Входит в:** Осознанность взрослого  
**Выход:** Свободнее от иллюзий, четче решения

## 🔗 КАК PACK "СИДИТ" В БОЛЬШУЮ СИСТЕМУ

```
ZP (Нулевые принципы)         ← Фундамент
        ↑
       FPF (Первые принципы + фреймворк)  ← Основание
        ↑
        SPF (Форма для вторых принципов)  ← Обреколи
        ↑
   KIDS-PACK (Вторые принципы для детей) ← МЫ ЗДЕСЬ
        ↑
   Конкретная семья (3-й принципы)       ← Применение
```

**Логика:**
- Если ZP нарушаю → пересматриваю основание (FPF)
- Если FPF не подходит → пересматриваю форму (SPF)
- Если SPF не работает → пересматриваю содержание (KIDS-PACK)
- Если KIDS-PACK не работает → адаптирую для своей семьи

## 📈 ОЖИДАЕМАЯ ТРАЕКТОРИЯ (типичный случай)

```
Месяц 1:
- Взрослый учится держать различения
- Ребёнок начинает замечать (мини-фокус)
- Результат: ~5–10% спонтанного применения

Месяц 2:
- Взрослый спокойнее, вопросы идут "автоматом"
- Ребёнок начинает инициировать ("давай проверим?")
- Результат: ~30–40% спонтанного применения

Месяц 3:
- Взрослый видит паттерны, адаптирует
- Ребёнок уже применяет в новых контекстах БЕЗ подсказки
- Результат: ~60–80% спонтанного применения, начало переноса
```
